# Full-Stack-Web-Application

A responsive Full-Stack Web Application developed as part of the Cognifyz Full Stack Development Internship. The project demonstrates frontend development, backend API creation, MongoDB integration, user authentication, external API integration, and server-side middleware.

## Features

- Responsive landing page using HTML, CSS, and Bootstrap
- Contact form with client-side validation
- Password strength indicator
- Express.js backend
- MongoDB database integration using Mongoose
- User Registration and Login
- Password hashing using bcrypt
- JWT Authentication
- GitHub API integration
- Request logging using Morgan
- Rate limiting using Express Rate Limit
- REST API endpoints
- Clean and responsive UI

## Installation

Clone the repository

```bash
git clone https://github.com/SharunKumarD/Full-Stack-Web-Application.git
```

Move into the project folder

```bash
cd Full-Stack-Web-Application
```

Install dependencies

```bash
npm install
```

Start MongoDB locally.

Run the application

```bash
node app.js
```

Open your browser and visit

```
http://localhost:3000
```
